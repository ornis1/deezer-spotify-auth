# app

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Set your environment variables in .env file

- DEEZER_APP= ""
- DEEZER_SECRET= ""

- SPOTIFY_SECRET= ""

### Express server for forwarding requests

```
npm run backend
```

```
npm run build
```

### Lints and fixes files
